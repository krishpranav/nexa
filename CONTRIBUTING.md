# Contributing to Nexa

We welcome contributions! Please follow these guidelines:

1.  **Fork and Clone**: Fork the repo and clone it locally.
2.  **Branch**: Create a feature branch.
3.  **Code Style**: Use `cargo fmt` and ensure strict Rust 2024 compliance.
4.  **Tests**: Add tests for new features. Run `cargo test` before submitting.
5.  **Clean Commit History**: Squash commits if necessary.
6.  **Pull Request**: Submit a PR with a clear description of changes.

## Development Setup

1.  Install Rust (latest stable).
2.  Install `winit` dependencies for your platform.
3.  Run `cargo check --workspace` to verify setup.
